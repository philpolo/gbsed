### Python setup
All of our experiments were performed on an Ubuntu 24.04 system with a Python 3.12 environment.

**Create a python environment**
```bash
conda create --name av python=3.12
```

**Activate the new python environment**
```bash
conda activate av
```
**Requirements installation**

Once the environment activated, install the rest of the requirements from the requirements.txt
```bash
pip install -r requirements.txt
```
